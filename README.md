<div>
  
    <h2 align="center">
        <a name="projeto"></a> Projeto 
    </h2>
  
    Criar um calculado com funções de:
      
    - Soma
      
     - Subtração
     
     - Divisão
     
     - Multiplicação
      
     - Exponenciação
     
     - Porcentagem
     
     - Função "Apagar" (para corrigir o último dígito)
     
     - Função "Limpar" (para zerar o cálculo)
     
</div>
